## Church Account / Gospel Library finding (Command C-D)

No official, supported public developer API or Church Account OAuth path for
third-party apps exists. The Church's tech forum states no public APIs are
provided; gospel-content licensing goes through a permissions agreement
(permissions.churchofjesuschrist.org, contact tech@ldschurch.org); community
APIs (LDSorg/lds.io, LCR-API) are unofficial and unsupported. **Decision:
no sign-in implemented; public official links remain the production path.**
Sources: tech.churchofjesuschrist.org forum threads t=30587, t=39793, t=31409;
lds365.com third-party developer access note; github.com/LDSorg.
